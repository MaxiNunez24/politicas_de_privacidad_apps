# politicas_de_privacidad_apps
Este repositorio contiene las políticas de privacidad a utilizar en mis aplicaciones de Android, solicitado por Google Play

## Contenido

La página de política de privacidad está disponible en `index.html` y contiene el siguiente texto:

"Esta aplicación no recopila ni comparte información personal del usuario. Los datos se almacenan únicamente en el dispositivo del usuario."

## GitHub Pages

Para habilitar GitHub Pages y hospedar esta página:

1. Ve a la configuración del repositorio (Settings)
2. En el menú lateral, selecciona "Pages"
3. En "Source", selecciona la rama que deseas usar (por ejemplo, `main` o `copilot/create-privacy-policy-page`)
4. Selecciona la carpeta raíz `/` (root)
5. Haz clic en "Save"

Una vez habilitado, la página estará disponible en:
`https://maxinunez24.github.io/politicas_de_privacidad_apps/`
